What is astro?

Astro is an AIO proxy site for all your needs including a chatbox, games, apps, and settings!

When was Astro created?

The first version of astro, was actually not called astro. It was called Rocket Unbl*cker. It was a simple proxy for my school. It existed when I founded Galactic Network, previously JServices. I wanted my site to be up there so it was decent. My first blowup was with Jordan's Math Work v3 and rocket unbl*cker v3. It got popular fast. I decided to rebrand rocket unbl*cker into galactic. I didn't have anywhere to host so I stopped development. Then, I decided to make rebrand galactic into astro and make astro static. Astro was static for a while, but it had a bunch of issues. So i found a host and now here I am.
TLDR: Astro is an old proxy rebranded and now new.

How do I host astro myself?

Look at GN Docs or go to Astro's Github.

What is the error 'No Bare Clients'?

It's built on other common problems, but it's most commonly an issue with the proxy failing to load. Please reload the page, and if the problem keeps happening, make a Github issue!

What is a proxy?

A web proxy allows users to browse the internet anonymously by acting as an intermediary between their device and the websites they visit.
